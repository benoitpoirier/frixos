Available Tokens:

  • [device] - Device name
  • [greeting] - Time-based greeting (Good Morning/Afternoon/Evening/Night)
  • [day] - Current day of the week
  • [date] - Current date
  • [mon] - Current month
  • [temp] - Current temperature
  • [hum] - Current humidity
  • [high] - Today's high temperature
  • [low] - Today's low temperature
  • [rise] - Sunrise time
  • [set] - Sunset time
  • [wind] - Wind speed and direction
  • [gust] - Wind gust speed and direction
  • [precip] - Precipitation amount and probability
  • [uv] - UV index
  • [pressure] - Sea-level pressure with trend
  • [3high] - Highest temperature across 3 days
  • [3low] - Lowest temperature across 3 days
  • [HA:entity_id:path] - Fetch settings from Home Assistant (needs enabled integration)
  • [$:symbol] - Fetch stock price quote (needs enabled stock integration with Finnhub.io)
  • [CGM:glucose] - Latest glucose reading from your CGM device, formatted, e.g. 157 mg/dl (+3 @ 12:01). (needs active CGM integration)
  • [CGM:reading] - Latest glucose reading, plain, e.g. 157

Note: Tokens are case-sensitive and must be enclosed in square brackets.

Default message: [device].local/ [greeting] [day], [date] [mon], now [temp] today [high]-[low], hum. [hum], sun [rise]-[set]

Upload Firmware / Files

Your device checks and installs new firmware automatically.

If you have been instructed by our support to manually install a specific firmware version, or if you are looking to upload e.g. a custom font, you can do this here.


Select a firmware file (.bin) or other file to upload to the device.

0%

Firmware Update Instructions

  1. Click the "Choose File" button and select the firmware file (.bin) from your computer.
  2. Click the "Upload Firmware" button to start the update process.
  3. Wait for the update to complete. Do not power off or disconnect the device during the update.
  4. The device will automatically restart after the update is complete.

Warning: Uploading incorrect firmware may cause the device to malfunction. Make sure to use the correct firmware for your device.

User file upload instructions

  1. Click the "Choose File" button and select the file (e.g. user-font.jpg) from your computer.
  2. Click the "Upload Firmware" button to start the upload process.
  3. Wait for the upload to complete. Do not power off or disconnect the device during the upload.
  4. You can now upload another file or close this window.
